Click Counter — Extensão Chrome

Sobre a extensão

Click Counter é uma extensão simples para o Google Chrome que conta os cliques do usuário. Cada clique gera partículas animadas no popup!

Instalação manual

  1. Acesse chrome://extensions no Chrome.
  2. Ative o Developer mode no canto superior direito.
  3. Clique em Load unpacked e selecione a pasta do projeto.
  4. Abra o popup pelo ícone da extensão e teste os cliques.
