Click Counter

Extensão Chrome MV3 — Contador de cliques persistente.

Como instalar (modo desenvolvedor)

  1. Acesse chrome://extensions e ative Developer mode.
  2. Clique em Load unpacked e selecione a pasta do projeto.

Depois de publicar uma Release no GitHub, aqui aparecerá o link para o .zip.
